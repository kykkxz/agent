#### 3.2 缺失值的处理方法

缺失值处理的“四步法”

- 1.丢弃法：
  
  dp.dropna()：只要改行有空值就直接删除
  
  dp.dropna(thresh=3):只有当一行中有至少3个非空值才进行保留

- 2.固定值填充：
  
  df.fillna(0), df.fillna("未知“)。适用于明确缺失代表一种状态状态

- 3.统计值来进行填充：
  
  -均值：数据分布比较均匀，没有极端异常值的数据
  
  -中位数：**最推荐**。能够代表大部人的水平
  
  -众数：适合分类的数据

- 4.分组填充（进阶）：不同组别的统计值进行填充。薪资空缺：比如北京和三线城市，薪资差距巨大，需要按城市分组，用该城市的中位数来填补

```
df['salary'].fillna(df['salary'].median()) #中位数
df['salary'].fillna(df['salary'].mean()) #平均数
df['education'].fillna(df['salary'].mode()[0]) #众数
df['education'].fillna("未知) #固定值
df['salary'] = df.groupby(['city', 'position'])['salary'].transform(lambda x: x.fillna(x.median())) #按分组的统计值进行填充
```

#### 3.3 异常值的检测与清洗

1.什么是异常值？

不符合常识或者统计规律，例如：年龄999，身高3.5米

2.核心检测方法：IQR（四分位距法）

* **Q1 (下四分位)**: 25% 的数据小于这个值。
* **Q3 (上四分位)**: 75% 的数据小于这个值。
* **IQR = Q3 - Q1**
* **正常范围**：`[Q1 - 1.5 * IQR, Q3 + 1.5 * IQR]`

```python
import pandas as pd

df = pd.DataFrame({
    "name": ["张三", "李四", "王五", "赵六", "王二","钱八"],
    "salary": [4000, 4500, 5000, 5500, 6000, 100000]  # 100000 是明显的高薪异常
})

print("原始数据:")
print(df)

# 1. 计算四分位
Q1 = df["salary"].quantile(0.25)
# print(f"Q1 (下四分位): {Q1}")
Q3 = df["salary"].quantile(0.75)
# print(f"Q3 (上四分位): {Q3}")
IQR = Q3 - Q1
print(f"IQR: {IQR}")

# 2. 确定正常范围边界
lower_bound = Q1 - 1.5 * IQR
upper_bound = Q3 + 1.5 * IQR

print(f"\n📊 统计信息:")
print(f"Q1 (下四分位): {Q1}")
print(f"Q3 (上四分位): {Q3}")
print(f"正常范围: {lower_bound} ~ {upper_bound}")

# 3. 找出异常值
outliers = df[(df['salary'] < lower_bound) | (df['salary'] > upper_bound)]
print("\n🔍 发现的异常值:")
print(outliers)

# 4. 处理异常值
print("\n请选择处理方式：")
print("1. 直接删除（数据量充足时）")
print("2. 用中位数替换（数据量珍贵时）")

# 演示方法 B: 用中位数替换（避免数据量骤减）
df.loc[(df['salary'] < lower_bound) | (df['salary'] > upper_bound), 'salary'] = df['salary'].median()
#df.loc[行条件, 列名称] #按标签取行，对应列的值
#(df['salary'] < lower_bound) | (df['salary'] > upper_bound) # &,|
print("✅ 异常值处理完成 (替换为中位数):")
print(df)
```

```python
import pandas as pd
import numpy as np

# 构造脏数据
df = pd.DataFrame({
    "姓名": ["张三", "张三", "李四", "王五", "赵六",np.nan,"钱八"],
    "籍贯": ["北京", "北京", "上海", "京", "北京","深圳","北京"],
    "年龄": [25, 25, 150, -5, 28,25,30],
    "薪资": [8000, 8000, 12000, 20000, 15000,10000,"10k"]
})

print("原始脏数据:")
print(df)

# 1. 去重
# print(df.duplicated())
df = df.drop_duplicates()
print("去重后数据:")
# print(df)

# 2. 删除缺失值
df = df.dropna(subset=['姓名'])
# print("删除缺失值后数据:")
# print(df)

# 3. 删除异常值
# print(((df['年龄'] < 0) | (df['年龄'] > 120)))
df = df[(df['年龄'] >= 0) & (df['年龄'] <= 120)]
print("删除异常值后数据:")
print(df)

# 4. 把 "10k" 替换成 "10000" (处理特殊字符)
df['薪资'] = df['薪资'].astype(str)
df['薪资'] = df['薪资'].str.replace('k', '000')
df['薪资'] = df['薪资'].astype(int)
print("处理后薪资:",df['薪资'])

# 5. 处理籍贯
df.loc[df["籍贯"].str.contains("京", na=False), "籍贯"] = "北京"
#df.loc[行条件,列名称]
print("完成所有清洗步骤后的数据:")
print(df)
```
